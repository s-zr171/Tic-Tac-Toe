# Tic-Tac-Toe Game (Java Swing)

## 📌 Project Overview
This project is a Java-based implementation of the classic Tic-Tac-Toe game with a graphical user interface built using Java Swing. The game allows two players to compete locally by taking turns as Player X and Player O. It includes real-time game status updates, win detection, score tracking, and visual feedback for winning combinations.

The project was developed to strengthen understanding of object-oriented programming concepts, event-driven programming, and GUI development in Java.

---

## 🎮 Features
- Graphical user interface using Java Swing
- Two-player local gameplay (Player X vs Player O)
- Turn-based logic with status updates
- Automatic win and draw detection
- Scoreboard tracking wins for both players
- Highlighting of winning combinations
- Restart game functionality

---

## ⚙️ Technologies Used
- Java
- Java Swing (JFrame, JButton, JLabel, JPanel)
- Object-Oriented Programming (OOP)
- Event handling with ActionListeners
